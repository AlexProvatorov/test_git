BASH

1)Как перейти в домашний каталог из любого другого?
    cd ~
2)Как создать новый текстовый файл 123.txt в текущем каталоге?
    touch 123.txt
3)Как создать новую папку 1234 в текущем каталоге?
    mkdir 1234
4)Как создать в каталоге /abc ссылку на файл 123.txt из домашнего каталога?
    mkdir abc && cd mkdir
    sudo ln -s ../123.txt
5)Как создать в каталоге /abc символьную ссылку на папку 1234 из домашнего каталога?
    sudo ln -s ../1234
6)Как скопировать файл 123.txt из домашнего каталога в корневой?
    cd ~
    sudo cp 123.txt /
7)Как переместить файл 123.txt из домашнего каталога в каталог на 2 уровня выше?
    sudo mv 123.txt ../..
8)Как переименовать файл 123.txt в файл txt.123
    mv 123.txt txt.123
9)Как удалить файл 123.txt
    rm 123.txt
10)Как удалить все файлы и папки из каталога /abc, не удаляя сам каталог?
    rm -r abc/*
11)Как удалить все файлы и папки из какого-нибудь каталога без всплывающих вопросов?
    rm -r -f abc/*
12)Как переместиться в другую папку (abc) родительского каталога (соседнюю по отношению к текущей)?
    mv -r abc abc2
13)Как посмотреть в каком каталоге мы сейчас находимся?

24)Что делает команда tail? Какие у нее есть флаги?
    выводит последние 10 файлов, а head первые 10
    флаги:

    -c - выводить указанное количество байт с конца файла;
    -f - обновлять информацию по мере появления новых строк в файле;
    -n - выводить указанное количество строк из конца файла;
    --pid - используется с опцией -f, позволяет завершить работу утилиты, когда завершится указанный процесс;
    -q - не выводить имена файлов;
    --retry - повторять попытки открыть файл, если он недоступен;
    -v - выводить подробную информацию о файле;

    tail -f file_name отслеживаем логи

26)Как работает аутентификация по открытому / закрытому ключам?
    Аутентификация с открытым ключом SSH опирается на асимметричные
    криптографические алгоритмы, которые генерируют пару ключей, один
    из которых является «закрытым», а другой - «открытым». Вы храните
    закрытый ключ в секрете и храните его на компьютере, который 
    используете для подключения к удаленной системе.

27)Где хранятся открытые ключи на удаленном сервере?
    По умолчанию ключи хранятся в каталоге ~/. ssh внутри домашнего каталога 
    вашего пользователя. Закрытый ключ будет иметь имя id_rsa , а 
    соответствующий открытый ключ будет иметь имя id_rsa.
